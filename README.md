# apk-utils

A collection of tools and checklists for Android bug hunting.

## Structure

    apk-utils/
    ├── scripts/
    │   └── pull-base-apk.sh
    ├── docs/
    │   └── pull-base-apk.md
    ├── README.md
    └── LICENSE

## Scripts

| Script                                      | Description                                                          |
| ------------------------------------------- | -------------------------------------------------------------------- |
| [`pull-base-apk.sh`](docs/pull-base-apk.md) | Search for an installed package and pull only its `base.apk` via ADB |

More scripts and checklists will be added here as the collection grows — see individual docs above for requirements and usage.
